# Shorts Studio — MINDLOOP

Інструменти для вірусного каналу коротких відео (TikTok + YouTube Shorts)
у 4 мовах: укр / рос / англ / ісп.

## Файли

| Файл | Призначення |
|------|-------------|
| `CONTENT_PLAN.md` | Дослідження алгоритмів 2026, формула, промт, план на 30 днів |
| `generate_video.py` | Генератор відео (9:16, 1080×1920, 33 с, 4 мови) |
| `upload_youtube.py` | Автозавантаження на YouTube через офіційний API |

## Швидкий старт

```bash
pip install pillow numpy            # + ffmpeg у системі
python3 generate_video.py all out/  # рендерить 4 мовні версії (~2 хв)
```

Новий сюжет = замінити словник `CONTENT` у `generate_video.py`
(промт для генерації текстів — у `CONTENT_PLAN.md`, розділ 2).

## Створення каналів (вручну, ~5 хв на платформу)

- **YouTube:** увійти в Google → [studio.youtube.com](https://studio.youtube.com)
  → створити канал. Окремий канал на кожну мову.
- **TikTok:** реєстрація в застосунку. Окремий акаунт на мову
  (@mindloop.ua, @mindloop, @mindloop.es...). TikTok геотаргетує за мовою
  та регіоном SIM/IP — акаунт для іспанської аудиторії краще вести
  з відповідним таргетингом.

## Публікація

- **YouTube:** вручну або `python3 upload_youtube.py video.mp4 "Назва" "Опис" "теги"`
  (потрібен OAuth-ключ — інструкція в шапці скрипта).
- **TikTok:** вручну через застосунок (⚠️ додайте трендовий звук на 20–30%
  гучності). Content Posting API TikTok потребує окремого схвалення
  розробницького акаунта — для старту простіше вручну.
