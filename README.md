# ARMADA HR — сайт вакансій

Статичний сайт агенції ARMADA HR: працевлаштування українців у готелях Польщі (Щирк і Вісла, готелі Mercure Szczyrk Resort та Aries Hotel & Spa).

**Сторінки:**
- `index.html` — головна: вакансії, умови, контакти, банер «Шукаємо кухарів»
- `privacy.html` — політика конфіденційності (вимога Meta для реклами/додатків)
- `data-deletion.html` — інструкція видалення даних (вимога Meta)

**Технології:** чистий HTML + CSS, без JS, без збірки, без залежностей.

**Запуск локально:** відкрити `index.html` у браузері — цього достатньо.

**Повна документація:** див. [DOCS.md](DOCS.md) — архітектура, аудит, база знань проєкту.
